# GESTION-DE-CALIFICACIONES-USANDO-ARREGLOS

Mi programa permite capturar datos usando arreglos y estructuras repetitivas como: (for, while o do-while),
procesar información usando estructuras de control y mostrar resultados en pantalla de forma clara de manera que se puedan visualisar las calificaciones de cada estudiante de manera ordenada y organizada ademas de calcular el promedio, la calificacion mas alta y la mas baja.

En el código usé arreglos para poder guardar todas las calificaciones sin tener que crear muchas variables. El arreglo `calificaciones[20][4]` lo usé para guardar las 4 notas de cada estudiante, donde la primera posición es el número del estudiante y la segunda es cada una de sus calificaciones, así puedo ir guardando y luego mostrar cada nota fácilmente con los ciclos. También usé otro arreglo llamado `promedioEstudiantes[20]` para guardar el promedio de cada estudiante después de sumar sus 4 notas. Con los ciclos `for` voy recorriendo esos arreglos para ingresar los datos, hacer los cálculos y después mostrarlos en pantalla de forma ordenada.
