# GESTION-DE-CALIFICACIONES-USANDO-ARREGLOS

Mi programa permite capturar datos usando arreglos y estructuras repetitivas como: (for, while o do-while),
procesar información usando estructuras de control y mostrar resultados en pantalla de forma clara de manera que se puedan visualisar las calificaciones de cada estudiante de manera ordenada y organizada ademas de calcular el promedio, la calificacion mas alta y la mas baja.
